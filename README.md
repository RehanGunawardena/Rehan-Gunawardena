# Rehan Gunawardhana — Academic Portfolio

A responsive, no-build GitHub Pages portfolio built in plain HTML, CSS, and JavaScript.

## Publish on GitHub Pages

1. Create a GitHub repository named `YOUR-USERNAME.github.io`.
2. Upload **all files and folders** from this project to the repository root.
3. In GitHub, open **Settings → Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select the `main` branch and `/ (root)` folder, then save.
6. The website will be published at `https://YOUR-USERNAME.github.io/`.

For a project repository (for example `portfolio`), follow the same steps; GitHub publishes it at `https://YOUR-USERNAME.github.io/portfolio/`.

## Before publishing

- Replace `https://github.com/` and `https://www.linkedin.com/` in `index.html` with your actual profile links.
- Add Google Scholar, ORCID, ResearchGate, or YouTube links when available.
- Replace the monogram portrait with a professional photograph if desired.
- Replace `assets/CV_Rehan.pdf` whenever the CV is updated. Keep the same filename so the Download CV button keeps working.

## Files

- `index.html` — all website content
- `style.css` — responsive visual design
- `script.js` — menu, navigation highlighting, scroll animation
- `assets/CV_Rehan.pdf` — downloadable CV

No installation, Jekyll, or build process is needed.
